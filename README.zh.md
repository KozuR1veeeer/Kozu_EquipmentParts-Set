# 河津重工零部件素材包

🌍 Languages: [日本語](README.md) | [English](README.en.md) | [中文](README.zh.md) | [한국어](README.kr.md)

## 简介
这是一个将至今为止制作的各类零部件素材汇总在一起的部件素材包。

## 内容清单
### 驾驶台
* **EV-E301系类型**
* **E129系类型(类似的东西)**：因为是7年前做的老古董，所以质量较低。

### 车顶设备
#### 空调
* **AU720**：搭载于209系等列车上的集中式空调装置
* **AU736**：搭载于EV-E301系上的集中式空调装置
* **CU708**：搭载于东急5000系列等列车上的集中式空调装置
* **HRB352-2**：搭载于阪急9000系上的分散式空调装置

#### 受电弓
* **PS38**：搭载于EV-E301系上的受电弓(直流型)
* **PS37A＋交直流设备**：搭载于E531系等交直流型电车上的受电弓   
                    <br>作为赠品，还附带了交直流型电车特有的复杂管线。

#### 其他
* **无线电天线**：JR东日本电车上常见的小型米色盒子

### 车下设备
#### 车下设备本体
* **209系类型**：模型为控制装置更新前的样式
* **京成3700形类型**：8年前做的老古董，设备布局和本身质量都很糟糕
* **东急5000系类型**：设备布局略有不同，计划以后重做
* **阪急1000系类型(类似的东西)**：为某处制作模型时的副产物　
                                  <br>VVVF也有All-SiC型的外壳版本

#### 转向架
* **DT61**：209系～E231系世代常见的轴梁式无摇枕转向架
* **DT79**：E233系以后的JR东日本电车上常见的轴梁式无摇枕转向架
* **FS-365**：东武8000系初期车等使用的明登·多伊茨式转向架
* **FS-396**：东武8000系后期车等使用的S明登式转向架
* **FS-579**：阪急1000系等使用的带摇枕单拉杆式转向架　
          <br>还有浅灰色专用的贴图哦
* **ND-718**：圆锥叠层橡胶式无摇枕转向架　~~老实说我不记得为什么要造这个了~~
* **TS-807B**：东急波纹车使用的轴弹簧式直接安装转向架
* **TS-1019**：东急5000系使用的轴梁式无摇枕转向架

#### 车钩
* **河津式密接车钩**：模型大概是参考了JR东日本一带的电车，可搭载1段/2段式电气车钩
* **KE97**：跳线插座　~~老实说我甚至不知道它到底像不像真的~~
