# 河津重工業部品素材セット

🌍 Languages: [日本語](README.md) | [English](README.en.md) | [中文](README.zh.md) | [한국어](README.kr.md)

## 概要
これまで作ってきた部品素材を一纏めにしたパーツセットです。


## おしながき
### 冷房装置
* **AU736**: EV-E301系に搭載されている冷房装置
* **CU708**: 東急5000系列に搭載されている冷房装置

### パンタグラフ
* **PS38**: EV-E301系に搭載されているパンタグラフ

### 運転台
* **EV-E301系タイプ**: 2048*2048基準で作ったのでテクスチャサイズがデカいです
* **E129系タイプ(っぽい何か)**: 7年前に作ったシロモノなのでクオリティ低いです

### 床下機器
* **209系タイプ**: モデルは制御装置更新前
*  **京成3700形タイプ**: 8年前に作ったシロモノなので機器配置とかそもそものクオリティが終わってます
*  **東急5000系タイプ**: 今回から新しくなりました
*   **阪急1000系タイプ(っぽい何か): 某所向けに作ったものの副産物　VVVFはAll-SiC型の筐体もあります

### 台車
* **DT79**: JR東日本でよく見る軸梁式ボルスタレス台車　DT71Bの代用品でも使えます
* **FS-365**: 東武8000系の初期車等で使用されたミンデンドイツ式台車
* **FS-396**: 東武8000系の後期車等で使用されたSミンデン式台車
* **FS-579**: 阪急1000系等で使われているボルスタ付きモノリンク式台車<br>
M車用/T車用、及び先頭台車用が収録されてます。あとライトグレー用のテクスチャもあるよ
