## 概要
これまで作ってきた部品素材を一纏めにしたパーツセットです。

## おしながき
### 運転台
* **EV-E301系タイプ**
* **E129系タイプ(っぽい何か)**: 7年前に作ったシロモノなのでクオリティ低いです

### 屋根上機器
#### クーラー
* **AU720**: 209系等に搭載されている集中式冷房装置
* **AU736**: EV-E301系に搭載されている集中式冷房装置
* **CU708**: 東急5000系列に搭載されている集中式冷房装置
* **HRB352-2**: 阪急9000系に搭載されている分散式冷房装置

#### パンタグラフ
* **PS38**: EV-E301系に搭載されているパンタグラフ(直流型)
* **PS37A＋交直流機器**: E531系等の交直流型電車に搭載されているパンタグラフ
                    <br>オマケで交直流型電車特有のゴチャゴチャ配管も付いてます

#### その他
* **ラジオアンテナ**: JR東日本の電車でよく見るちっちゃいベージュの箱

### 床下機器
#### 床下機器本体
* **209系タイプ**: モデルは制御装置更新前
* **京成3700形タイプ**: 8年前に作ったシロモノなので機器配置とかそもそものクオリティが終わってます
* **東急5000系タイプ**: 機器配置が若干違います　そのうち作り直す予定
* **阪急1000系タイプ(っぽい何か)**: 某所向けに作ったものの副産物　
                                  <br>VVVFはAll-SiC型の筐体もあります

#### 台車
* **DT61**: 209系～E231系世代でよく見られた軸梁式ボルスタレス台車
* **DT79**: E233系以降のJR東日本の電車でよく見る軸梁式ボルスタレス台車
* **FS-365**: 東武8000系の初期車等で使用されているミンデンドイツ式台車
* **FS-396**: 東武8000系の後期車等で使用されているSミンデン式台車
* **FS-579**: 阪急1000系等で使用されているボルスタ付きモノリンク式台車
              <br>ライトグレー用のテクスチャもあるよ
* **ND-718**: 円錐積層ゴム式ボルスタレス台車
* **TS-807B**: 東急のコルゲート車で使用されている軸ばね式ダイレクトマウント台車
* **TS-1019**: 東急5000系で使用されている軸梁式ボルスタレス台車

#### 連結器
* **河津式密着連結器**: モデルは確かJR東日本辺りの電車　1段/2段式電気連結器搭載可能
* **KE97**: ジャンパ栓
