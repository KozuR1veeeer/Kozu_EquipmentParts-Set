# 河津重工業部品素材セット

🌍 Languages: [日本語](README.md) | [English](README.en.md) | [中文](README.zh.md) | [한국어](README.kr.md)

## 概要
これまで作ってきた部品素材を一纏めにしたパーツセットです。


## おしながき
* **K703編成**: 川崎重工製 / 現行仕様 / シングルアームパンタグラフ / LED式行先表示器
* **K704編成**: 東急車輛製 / 現行仕様 / シングルアームパンタグラフ / 幕式行先表示器

## 主な機能
* **室内灯**: RTM標準の室内灯機能を使用して室内灯を点灯させることができます。OFFで消灯、ONで点灯、Rainbowで光度50%で点灯を切り替えられます
* **運番表示器**: KaizPatchのカスタムボタン機能を使用し、5桁の運行番号を個別に設定できます。
